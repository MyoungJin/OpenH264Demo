# OpenH264Demo
OpenH264 解码H264裸流文件示例。

## 目录结构
```
OpenH264Demo.sln      VS工程
OpenH264Demo.vcxproj  VS项目
src/*                 项目源文件
src/openh264/*        OpenH264项目文件
src/720p.h264         H264裸流
src/720p.yuv          YUV(I420)裸流，项目运行成功后的输出文件
libs/*                OpenH264静态链接库
```

## 编译环境
Winodws 7/8/10
Visual Studio 2013
